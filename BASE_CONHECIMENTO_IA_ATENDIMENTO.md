## 2. Como Fazer Pedido

### 2.1 Pelo Site (recomendado)

1. Acesse o site: https://heloconfeitarianr.web.app
2. Navegue pelo catálogo de produtos (abas de menu no topo)
3. Clique em "Adicionar" nos produtos desejados
4. Abra o carrinho (ícone no canto superior direito)
5. Preencha seus dados: nome completo e WhatsApp para contato
6. Escolha a forma de entrega: retirada ou entrega
7. Se escolher entrega, preencha: rua, bairro e ponto de referência
8. Escolha a data e hora desejada (em modo evento)
9. Selecione a forma de pagamento: PIX, dinheiro ou cartão
10. Clique em "Finalizar Pedido"
11. Você será redirecionado para o WhatsApp com uma mensagem pré-formatada contendo todos os dados do pedido
12. Envie a mensagem para confirmar

Importante: Todos os pedidos devem ser feitos através do site. Não aceitamos pedidos por telefone.

## 3. Formas de Pagamento

### 3.1 PIX

A chave PIX está disponível no site. Como pagar:
1. Copie a chave PIX no site
2. Abra o app do seu banco
3. Cole a chave e confirme o pagamento
4. Envie o comprovante pelo WhatsApp

### 3.2 Dinheiro

Pagamento na entrega ou retirada. É obrigatório informar o troco no carrinho. Leve valor exato se não precisar de troco.

### 3.3 Cartão

Aceitamos cartão na entrega e na retirada. Parcelamento disponível em até 3x. As taxas são calculadas automaticamente no carrinho.

## 7. Dúvidas Frequentes (FAQ)

### 7.1 O pedido é confirmado automaticamente?

Não. Após finalizar no site, você é redirecionado para o WhatsApp com uma mensagem pré-formatada. O pedido só é confirmado após você enviar essa mensagem e receber confirmação da confeitaria.

### 7.2 Posso alterar meu pedido após enviar?

Sim, entre em contato pelo WhatsApp o mais rápido possível. Alterações podem não ser possíveis se o pedido já estiver em produção.

### 7.3 Quanto tempo demora para o pedido ficar pronto?

O prazo é informado ao confirmar o pedido.

### 7.4 Vocês entregam em qualquer bairro?

Sim, entregamos em Nova Russas e região.

### 7.5 Posso pagar com cartão na entrega?

Sim, aceitamos cartão na entrega e na retirada, com parcelamento em até 3x.

### 7.6 O que acontece se eu não estiver no horário da entrega?

Se não estiver no endereço no horário combinado, o motorista fará tentativas de contato. Após isso, o pedido pode ser devolvido.

### 7.7 Vocês fazem encomendas personalizadas?

Sim. Entre em contato pelo WhatsApp para discutir encomendas especiais.

### 7.8 Os produtos contêm alergênicos?

Informamos sobre alergênicos na descrição de cada produto. Se tiver alergia específica, entre em contato para confirmar.

### 7.9 Vocês têm opções sem glúten/sem lactose?

Sim, temos opções sem glúten e sem lactose. Entre em contato para mais informações.

### 7.10 Como funciona o programa de fidelidade?

### 9.4 Capacidades de Ação (Function Calling)

Você tem acesso a ferramentas que permitem executar ações no site em nome do cliente. Use essas ferramentas quando apropriado para melhorar a experiência do usuário.

#### Ferramentas Disponíveis

**1. adicionar_ao_carrinho**
- **Quando usar:** Quando o cliente pedir para adicionar um produto específico ao carrinho (ex: "quero 2 bolos de chocolate", "adiciona brigadeiro", "me coloca um pudim")
- **Parâmetros:**
  - `produtoId` (string): ID do produto no catálogo (ex: "p1", "p2")
  - `nomeProduto` (string): Nome do produto para confirmação (ex: "Bolo de Chocolate")
  - `quantidade` (number): Quantidade a adicionar (padrão: 1)
- **Fluxo recomendado:**
  1. Se o cliente mencionar apenas o nome, primeiro use `consultar_estoque` para encontrar o produto pelo nome
  2. Se houver múltiplos produtos com nome similar, peça ao cliente para especificar
  3. Use `adicionar_ao_carrinho` com o ID correto
  4. Confirme ao cliente que o produto foi adicionado

**2. consultar_estoque**
- **Quando usar:** Quando o cliente perguntar sobre disponibilidade de um produto (ex: "tem bolo?", "tem brigadeiro?", "quantos pudins tem?")
- **Parâmetros:**
  - `produtoId` (string, opcional): ID do produto para consulta direta
  - `nomeProduto` (string): Nome do produto para busca
- **Fluxo recomendado:**
  1. Se tiver o ID, use-o para consulta direta
  2. Se tiver apenas o nome, a ferramenta busca por similaridade
  3. Informe o cliente sobre disponibilidade e limite de estoque (se houver)

#### Regras de Uso

- **SEMPRE** consulte o estoque antes de adicionar ao carrinho se não tiver certeza da disponibilidade
- Se o cliente pedir "quero tudo" ou "um de cada", pergunte quais produtos específicos deseja
- Se a ferramenta retornar erro (produto não encontrado), sugira ao cliente verificar o cardápio no site
- Após adicionar ao carrinho, confirme a ação e pergunte se deseja mais algo
- **NÃO** use ferramentas para finalizar pedidos - isso deve ser feito manualmente pelo cliente no site
- **NÃO** use ferramentas para remover itens do carrinho - oriente o cliente a fazer isso no site

#### Exemplos de Interação

**Cliente:** "Quero 2 bolos de chocolate"
**IA:** [Usa consultar_estoque com nomeProduto="Bolo de Chocolate"] → [Usa adicionar_ao_carrinho com produtoId, nomeProduto, quantidade=2] → "Adicionei 2x Bolo de Chocolate ao seu carrinho! Mais alguma coisa?"

**Cliente:** "Tem brigadeiro?"
**IA:** [Usa consultar_estoque com nomeProduto="Brigadeiro"] → "Sim, temos Brigadeiro disponível! Quer que eu adicione ao seu carrinho?"

**Cliente:** "Quero tudo"
**IA:** "Posso ajudá-lo a adicionar produtos específicos ao carrinho. Quais produtos você gostaria de adicionar? Pode me dizer os nomes ou verificar o cardápio no site."
