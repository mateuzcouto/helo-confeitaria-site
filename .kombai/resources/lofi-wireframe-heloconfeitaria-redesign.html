Key Mobile Changes

[Issue #24/#25] Hero height
Reduce .hero-premium mobile padding-bottom from 2.8rem → 0.5rem. Saves ≈40px, moves search bar above the fold.
[Issue #28] Safe-area FAB
Add viewport-fit=cover to meta viewport.
Change: bottom: calc(0.8rem + env(safe-area-inset-bottom, 0px))
[Issue #27] Safari input zoom
Set all cart inputs to font-size: 16px to suppress iOS Safari auto-zoom on focus.
[Issue #26/#30] Touch targets
Category pills: padding: 11px 16px → 44px
Details btn: width: 44px; height: 44px
[Issue #29] Cart scroll bleed
Add overscroll-behavior: contain to cart .hide-scrollbar div. Prevents iOS rubber-band scroll from leaking to background page.
[Issue #23/#32] Brand colors
Fix theme-color: #8B4513 → #2A3D5D
Add html { background-color: #2A3D5D } to prevent white overscroll flash.
